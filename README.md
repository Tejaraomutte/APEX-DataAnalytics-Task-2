# Exploratory Data Analysis (EDA) & Business Intelligence Dashboard

## 📌 Project Overview
This project focuses on performing **Exploratory Data Analysis (EDA)** and creating a **Business Intelligence Dashboard Mock-up** using a laptop market dataset.  

The main objective of this project is to uncover patterns, trends, and relationships within the data and present meaningful insights through visualizations and dashboard reporting.

---

# 📂 Dataset Used
- **Dataset Name:** laptop_scrap_data.csv
- **Source:** Kaggle Laptop Dataset

The dataset contains information about:
- Laptop Brands
- Prices
- Ratings
- Processor Types
- Operating Systems
- Laptop Categories
- RAM & Storage configurations

---

# 🎯 Objectives
- Perform descriptive statistics and univariate analysis
- Analyze relationships between different variables
- Create visualizations for better understanding
- Identify key business insights
- Build a static dashboard mock-up for KPI tracking

---

# 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- PowerPoint

---

# 📊 EDA Tasks Performed

## 1. Data Cleaning
- Removed missing values
- Checked duplicate records
- Converted columns into proper formats
- Standardized dataset values

## 2. Descriptive Statistics
- Average laptop price
- Brand distribution
- Rating analysis
- Processor analysis

## 3. Data Visualization
Created:
- Bar Charts
- Histograms
- Pie Charts
- Correlation Heatmaps

## 4. Business Intelligence Dashboard
Designed a static dashboard mock-up containing:
- Total Models
- Average Price
- Average Rating
- Total Brands
- Brand Analysis
- Category Distribution
- Processor-wise Price Analysis

---

# 📈 Key Insights
- Dell and HP dominate the laptop market
- Apple laptops have the highest average price
- 8GB RAM laptops are the most common
- Gaming laptops occupy a significant market share
- Premium processors increase laptop pricing
